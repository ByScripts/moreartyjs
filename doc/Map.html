Class: Map

Map

HAMT-based persistent map implementation with String keys. Uses Java-style hashcode implementation.

Map methods try to return original instance if they don't 'change' it, e.g. on filter if no elements are filtered, on update if no value is changed, on join with empty map, and so on.
